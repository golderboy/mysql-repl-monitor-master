# R3.1.5-ws6 notes

ก้อนนี้ปิดงาน WS6 ตามเอกสาร Phase 3 โดยเน้น final sign-off, operations handoff และ closeout package

## สิ่งที่เพิ่ม
- `scripts/final_handoff_check.py` สำหรับตรวจว่า bundle ส่งมอบรอบปิดเฟสครบ
- operations handoff ฉบับ final
- ownership/contact matrix template
- closeout checklist
- final sign-off template
- backup/restore drill record template

## จุดประสงค์
- ให้ dev/IT/DBA/operator ใช้ baseline นี้ต่อได้โดยไม่ต้องเดา runtime layout
- ให้มีเอกสารที่กรอกต่อได้จริงตอนส่งมอบ
- ให้มีเครื่องมือตรวจว่า bundle final ยังครบและไม่หลุดไฟล์สำคัญ

## หมายเหตุ
- WS6 รอบนี้ไม่ได้เปลี่ยน route หลักหรือ worker หลัก
- ใช้เพื่อปิดงานส่งมอบและ baseline freeze ไม่ใช่ rewrite ระบบ
- ก่อน sign-off จริง ควรรัน `python scripts/final_handoff_check.py` พร้อม verify/test รอบล่าสุด
