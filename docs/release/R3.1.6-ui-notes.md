# R3.1.6-ui notes

เป้าหมายของแพตช์นี้คือปรับ UI ให้พร้อมทดสอบ browser flow ก่อนขึ้น production โดยไม่เปลี่ยน stack และไม่ rewrite baseline เดิม

สิ่งที่เพิ่ม
- ปรับ theme/CSS กลางและ main.js สำหรับ active nav + copy helper
- เพิ่มหน้า HTML ใหม่
  - /settings/advanced
  - /reports/daily
  - /jobs/<run_id> แบบ HTML
  - error pages 400/404/500
- เพิ่ม browser flow สำหรับ alerts create policy/target
- เพิ่มปุ่ม cancel job จากหน้า jobs และหน้า job detail
- ปรับ login/change-password/dashboard/reports/settings ให้ใช้งานและทดสอบง่ายขึ้น

ข้อจำกัด
- ใน environment ของ container นี้ไม่มี Flask dependency จึงไม่สามารถรัน verify_patch.py แบบ import app ได้
- ตรวจผ่านด้วย py_compile สำหรับไฟล์ Python ที่แก้แล้ว
- ให้รัน verify_patch.py และ unittest บนเครื่องเป้าหมายที่มี dependency ครบอีกครั้ง
