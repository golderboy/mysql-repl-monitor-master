# R3.1.1-ws2 notes

เป้าหมายของแพตช์นี้คือเก็บงาน P3-WS2 ฝั่ง operator workflow โดยไม่เปลี่ยน stack และไม่ชน route/API เดิม

สิ่งที่เพิ่ม
- role-aware landing หลัง login/logout/change password
- หน้า HTML สำหรับ /trends, /jobs, /alerts โดยยังคง JSON เดิมเมื่อเรียกแบบ API
- filter persistence สำหรับ incidents, deep-runs, trends, jobs
- reset filter flow ผ่าน ?reset=1
- navigation ลิงก์ไป Trends / Jobs / Alerts
- confirm step สำหรับ action เสี่ยงใน browser flow
- test เพิ่มสำหรับ login landing, trends HTML/JSON, jobs/alerts HTML

ข้อกำกับ
- ไม่แตะ app/config.py และ app/db.py
- ไม่เปลี่ยน public route path เดิม
- ไม่เปลี่ยน behavior ของ worker/cycle/backup
